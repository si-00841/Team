## 🌐 주제 : 해양 오염 쓰레기 현황

해양 쓰레기는 대부분 육지에서 유입되며, 해양 생태계뿐 아니라 경제와 관광에도 심각한 영향을 미칩니다.

- 📌 전 세계 해양 쓰레기 80%는 육상에서 유입
- 📌 매년 1,100만 톤의 플라스틱이 바다로 유입
- 📌 해양 플라스틱은 매년 4%씩 증가
- 📌 약 700종의 해양 생물이 피해를 입음
- 📌 해양 청소 쓰레기의 65%는 일회용품
- 📌 아태지역이 전체 해양 쓰레기의 50% 차지
- 📌 어업 피해 연간 약 25억 달러
- 📌 관광객 수 약 20% 감소
- 📌 미세플라스틱은 90% 이상 샘플에서 검출됨
- 📌 최근 5년간 국제 협약 30건 이상 체결
